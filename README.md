Para rodar o código tem que ter o Node.js instalado.
Node.js: Versão 20.19.0 ou superior
Um gerenciador de pacotes: npm
Instalação das Dependências
Abra o terminal na pasta raiz do projeto e execute o comando para instalar todos os pacotes necessários:

npm install

Após instalar as depenências, execute o comando abaixo:
npm run dev

O projeto será iniciado e ficará acessível no seu navegador (o endereço exato será mostrado no terminal).
